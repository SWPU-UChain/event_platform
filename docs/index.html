/event/{standard}/{category}/{page} get

a list of events

get

get a list of events

get /event/{standard}/{category}/{page}

get a list of events

  • Request
  • Response

URI Parameters

  • standard: required (string)

    media or type , 表明是按照报道媒体还是事件类别分类

    Examples:

    media
    type
  • category: required (string)

    报道媒体或事件类别名称

  • page: (integer - default: 1)

HTTP status code 200

get success

Body

Media type: application/json

Type: array of Event

Items: Event

  • id: required (string)

    unique identification of event

  • title: required (string)

    event's title

  • time: (integer)

    happened time of event, not required in relative_events

  • summarize: (string)

    summarize of event, not required in relative_events

  • influence: (number)

    influence of event, not required in relative_events

  • media: (array of )

    reported medias, not required in relative_events

Examples:

[
  {
    "id": "id1",
    "title": "title1",
    "time": 1483590060000,
    "summarize": "summarize1",
    "influence": 12.3,
    "media": [
      "weibo"
    ]
  },
  {
    "id": "id2",
    "title": "title2",
    "time": 1483590060000,
    "summarize": "summarize2",
    "influence": 12.5,
    "media": [
      "toutiao"
    ]
  }
]

comment info of an event

get

get /event/{id}/comment

get event's comment info

  • Request
  • Response

URI Parameters

  • id: required (string)

    unique identification of an event

HTTP status code 200

get comment info success

Body

Media type: application/json

Type: object

Properties
  • proportion: required (array of items)

    Items: items

    • type: required (string)

      emotion type

    • number: required (integer)

      number of this type

  • times: required (array of items)

    Items: items

    • word: required (string)

      word of comment

    • times: required (integer)

      number of this word

  • comments: required (array of items)

    Items: items

    • type: required (string)

      emotion type

    • comment: required (string)

      highest score comment in this type

  • dailyComments: required (array of items)

    Items: items

    • time: required (integer)

      timestamp of the day

    • number: required (integer)

      number of comments in the day

Examples:

{
  "proportion": [
    {
      "type": "喜",
      "number": 24
    },
    {
      "type": "惧",
      "number": 12
    }
  ],

    {
      "word": "精彩",

    },

      "word": "还行",
      "times": 38
    }
  ],
  "comments": [
    {
      "type": "喜",

    },
    {
      "type": "惧",
      "comment": "细思恐极"
    }
  ],
  "dailyComments": [
    {
      "time": 1483545600000,
